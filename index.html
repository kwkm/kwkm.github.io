memo

メソッドの動詞の使い分け

自分のための使い分けルール

生成系

add
登録する。
insert
SQLのinsert文を発行する用途。
register
オブジェクトに対して、振る舞いを変えるようなデータを登録する際に使用する。（出力時のフィルターを追加したり、ルーティングのルールを追加したり等）
set
プロパティに対して値を与える。
create
オブジェクトを生成して返す。

更新系

edit
編集する。
update
SQLのupdate文を発行する用途。
enable
オブジェクトに対して、特定の振る舞いを有効化する。
disable
オブジェクトに対して、特定の振る舞いを無効化する。

削除系

remove
削除する。
delete
SQLのdelete文を発行する用途。
clear
中身を空にする。
initialize
初期状態に戻す。
unregister
登録の解除

検索系

find
条件に一致した結果を取得する。
filter
条件によって除外された結果を取得する。
search
条件に一致するものを探す。
select
SQLのselect文を発行する用途。
